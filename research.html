IoT/Embedded Systems Security

針對物聯網與嵌入式系統單晶片，進行系統安全的研究。

在這個主題，首先要熟悉基本的 C 語言、作業系統 (Linux) 與密碼學相關的知識，並會接觸到大型的開源專案，進一步學會改寫專案的原始碼與 Makefile，並在專案上開發應用程式，甚至是撰寫 device driver 和 file system 等等 IC Design 公司所需要的系統技術。

我們目前研究的方向為嵌入式系統的 TEE (Trusted Execution Environment)，也就是 ARM TrustZone，此技術已套用在手機、車載晶片或物聯網裝置上，現今許多手機 OEM 與晶片大廠 (三星、高通、聯發科、新思) 都有引入。

  • TEE (Trusted Execution Environment, 中文介紹)
  • ARM TrustZone (技術簡介)
  • 主要開源專案：OP-TEE (GitHub 連結, 官方說明文件)
